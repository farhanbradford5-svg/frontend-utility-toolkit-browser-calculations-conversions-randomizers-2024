# CalcSpark

  A complete browser-based calculator and converter website with 250+ tools across 16 calculator categories, 11 converter categories, and randomizers.

  **Live site:** https://calcspark.replit.app

  ## Features
  - 250+ pure client-side calculators, converters & randomizers
  - No APIs, no data sent anywhere — everything runs in your browser
  - Categories: Math, Geometry, Statistics, Physics, Finance, Health, Date/Time, Cryptography, Construction, and more
  - Converters: Length, Weight, Temperature, Speed, Area, Volume, Data, Pressure, Energy, and more
  - Custom design with deep indigo + electric lime palette, Sora + Inter fonts

  ## Tech Stack
  - React + TypeScript + Vite
  - Wouter (client-side routing)
  - Lucide React (icons)
  - Pure CSS (no UI library)

  ## Getting Started
  ```bash
  pnpm install
  pnpm --filter @workspace/calcspark run dev
  ```
